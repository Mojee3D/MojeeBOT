MojeeBOT V1

Print It. Build It. Code It.

MojeeBOT V1 is an open-source, 3D-printable robot platform designed for makers, students, and educators. It brings together 3D printing, electronics, and coding into one hands-on project that’s easy to build and ready to expand.

---

What is MojeeBOT?

MojeeBOT V1 is a beginner-friendly robotics platform built using accessible components and fully 3D-printable parts.

At its core, it’s a small mobile robot that can:
- Drive using dual motors  
- Detect obstacles  
- Run custom programs  

But more importantly — it’s a platform.

You can modify it, expand it, and turn it into your own project.

---

Open Source

MojeeBOT V1 is fully open-source.

That means you can:
- Download and print all parts  
- Modify the design  
- Customize the code  
- Build your own version  

If you create something cool or improve the design, I’d love to see it.

---

What’s Included

This repository contains everything you need to build your own MojeeBOT:

- STL files for all printable parts  
- Bill of Materials (BOM)  
- Wiring guide  
- Build instructions  
- Starter code  

---

Hardware Overview

- **Controller:** Cytron Maker Pi RP2040  
- **Motors:** 2× N20 gear motors with encoders  
- **Sensor:** Ultrasonic distance sensor  
- **Power:** Battery-powered  
- **Chassis:** 3D printed round platform (~200mm)  

---

Build Overview

1. Print the parts  
2. Gather the components  
3. Assemble the chassis  
4. Install motors and wheels  
5. Wire the electronics  
6. Upload the code  
7. Start experimenting
